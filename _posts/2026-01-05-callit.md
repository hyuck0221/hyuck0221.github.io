## Call It 플러그인이란?

Jetbrains 기반 IDE 에서 사용할 수 있는 AI 변수명 생성 플러그인입니다<br>
Gemini API Key를 입력하여 빠른 변수명 작명이 가능합니다

### 지원 기능 (1.0 기준)

* 최신 Gemini 모델 사용 가능
* 5가지 다국어 지원
* 프로그래밍 언어 감지하여 맞춤 추천
* 동시 n개 변수명 추천

### 지원 IDE

Android Studio, AppCode, Aqua, CLion, Code With Me Guest, DataSpell, DataGrip, JetBrains Gateway, GoLand, IntelliJ IDEA, JetBrains Client, MPS, PhpStorm, PyCharm, Rider, RubyMine, RustRover, WebStorm, Writerside

## 설정

Settings > Tools > CallIt > Gemini API Key 입력 > 확인

![setting.png](../assets/images/post/callit/setting.png)

* Gemini API Key: google ai studio에서 발급받은 api key
* Gemini Model: 사용 가능한 모델들
* Max Suggestions: 추천 내역 수 <br>(1로 설정 시 즉시 적용)
* message Language: 설정 언어

## 사용법

변경하고 싶은 문장 드래그 우클릭 > Suggest Name with CallIt 클릭

![Suggest_photo_1.png](../assets/images/post/callit/suggest_photo_1.png)

원하는 제안 선택 후 OK<br>(Max Suggestions가 1이면 아래 창은 뜨지 않습니다)

![suggest_photo_2.png](../assets/images/post/callit/suggest_photo_2.png)
